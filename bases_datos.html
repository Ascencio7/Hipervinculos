Bases de Datos




¿Qué son las Bases de Datos?

Las Bases de Datos son espacios de almacenamiento de información organizados de manera estructurada y persistente, que permiten almacenar, recuperar, y modificar datos de manera segura.




Temas de Bases de Datos:

  • Modelos de datos
  • Normalización
  • Consultas SQL
  • Transacciones y control de versiones
  • Estructuras de datos y algoritmos
